SKYNovel開発環境　準備の手引き

SKYNovel 開発環境にエラーが ${エラー数}個あります。

　環境設定の手順を解説します。
  • 1.SKYNovel拡張機能の情報をチェック
  • 2.問題の解決・だいたいクリックする
  • 3.環境設定を再チェックする
  • 4.「windows-build-tools」の対応

1.SKYNovel拡張機能の情報をチェック

　VSCodeの Activity Bar に、SKYNovelの「丸に紙飛行機」マークが表示されているので、クリックして下さい。


　赤丸バツ印がエラーです。例では「Node.js（とnpm）」が見つからない、という状態を示します。


2.問題の解決・だいたいクリックする

　マウスカーソルをエラーメッセージの上に移動すると、何かしらのボタンが表示されます。
　クリックして下さい。


　ブラウザーが起動し、Node.js のインストーラーがダウンロードされます。開いて下さい。
（Windows）

（macOS）


　あとはウイザードに従ってインストールしていきます。（Windows / Mac 似たような画面なので、Windows のスナップショットで示します）
　基本は赤丸してあるボタンを押していったり、使用許諾・インストール先などを確認・指定していきます。
　特に変更せずそのままで進めても大丈夫です。










　この画面でインストールを待ちます。


　この画面が出た場合は「はい」を押して下さい。
　しばらくして、インストールが終了します。


3.環境設定を再チェックする

　「開発環境」のすぐ右にある「更新」ボタンをクリックして下さい。
　環境設定を再チェックします。


　エラーが消えました！


4.「windows-build-tools」の対応

　windowsのみ、もうひと手順あります。


　管理者権限で PowerShell を起動し、
　npm i -g windows-build-tools をペーストして実行（Enterボタン）します。


　文字がずらずら流れますが、それらが止まる（処理が終了する）まで待って下さい。


　その後再チェックボタンです。エラーが消えているはずです。
